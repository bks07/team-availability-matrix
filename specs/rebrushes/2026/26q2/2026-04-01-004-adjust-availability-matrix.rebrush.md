# Adjust Availability Matrix Layout

## WHAT
The availability matrix layout needs adjustments to improve alignment, spacing, and overall visual consistency.

### Changes:
1. **Inner Spacing**: The matrix should follow the same inner spacing as the toolbar card.
2. **Content Width**: The matrix should only use as much space as required by its content. Columns should be as narrow as possible.
3. **Centering**: If the matrix does not use the full width of the matrix card, it should be centered horizontally within the card.
4. **Header Fonts**: The "Date", "Day", and "Working" headers should use the same font and font size as the current "Day" header.
5. **Top Spacing**: Reduce the distance between the matrix card's upper border and the visible part of the matrix header table. The spacing should match the toolbar card's spacing.

## WHY
To ensure the availability matrix is visually consistent with other UI elements and optimally utilizes space, improving usability and aesthetics.

## ADDITIONAL INFORMATION
- Ensure the changes are responsive and maintain alignment across different screen sizes.
- Test the layout adjustments on various devices to confirm consistency.
