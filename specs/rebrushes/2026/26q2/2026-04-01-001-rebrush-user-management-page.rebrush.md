# Rebrush: User Management Page

## What
The user management page will be updated to improve usability and aesthetics:

1. **Table Width**: The user table will use the full width of the screen.
2. **Table Header Styling**: The table header will be visually distinct from the rest of the table.
3. **Action Buttons**: Replace `edit` and `delete` text buttons with icons. Hovering over these icons will display alt text with explanations.
4. **Create User Modal**: Replace the current inline user creation form with a `Create User` button that opens a modal for user creation.
5. **Search and Filter**: Add a search bar for user names and a filter dropdown for locations.

## Why
These changes aim to:
- Enhance the visual appeal and usability of the user management page.
- Save space by replacing the inline user creation form with a modal.
- Improve navigation and user management efficiency with search and filter options.
- Provide a cleaner and more modern interface.

## Additional Information
- The modal for user creation will include fields for email, display name, password, and optional location.
- The search bar will allow partial matches on user names.
- The location filter will list all available locations.
- The table header styling will use a distinct background color and bold text.
- Icons for `edit` and `delete` will follow a consistent design language with the rest of the application.
