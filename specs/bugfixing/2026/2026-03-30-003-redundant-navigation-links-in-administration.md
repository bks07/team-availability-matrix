# Redundant navigation links in administration area

## CURRENT BEHAVIOR

Although the application already provides a burger menu with the main navigation links, on the left side of the administration area are still the links to the admin pages.

## EXPECTED BEHAVIOR

The burger menu should provide access to the administration pages.

The additional navigation links on the left side of the administration area should no longer be shown.


## IMPACT

- The administration area shows duplicate navigation controls.
- Users are presented with two different navigation patterns for the same set of pages.
- The interface feels inconsistent because outdated links remain visible alongside the current navigation approach.
- The redundant links take up space in the administration area and add unnecessary visual clutter.


## STEPS TO REPRODUCE

1. Log in as an admin user.
2. Go to administration area.
3. Look at the left side of the viewport.
4. Observe that the page links to the admin pages are visible even though the burger menu already provides navigation links.

## ADDITIONAL INFORMATION

This issue affects the administration area layout.

The problem is not a lack of navigation. Instead, older navigation links are still displayed even though the burger menu already covers the same purpose.
