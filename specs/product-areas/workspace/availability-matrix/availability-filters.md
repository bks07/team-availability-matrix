# Availability Filters

## Story
- **IN ORDER TO** focus on specific availability statuses
- **AS** a user
- **I WANT TO** filter the availability matrix by status

## Acceptance Criteria
- Users can filter the matrix by W/V/A statuses.
- The matrix updates dynamically based on the selected filter.
- A reset button clears all filters.

## In-Scope
- Filter dropdown or buttons in the matrix UI.
- Backend support for filtered data.

## Out-of-Scope
- Advanced filtering options (e.g., by date range).

## Additional Information
- Filters should be intuitive and easy to use.
