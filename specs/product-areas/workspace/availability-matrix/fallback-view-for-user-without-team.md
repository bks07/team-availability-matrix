# Fallback View for User Without Team

## Story
- **IN ORDER TO** ensure users can still use the availability matrix
- **AS** a user without a team
- **I WANT TO** see a fallback view with only my availability column and a notification

## Acceptance Criteria
- The fallback view displays only the user's availability column.
- A notification is prominently displayed: "You are not yet part of a team. Join or create a team to collaborate."
- The fallback view is styled consistently with the team availability matrix.

## In-Scope
- Fallback view design and notification.
- Consistent styling with the team availability matrix.

## Out-of-Scope
- Team invitation or creation workflows.
- Advanced fallback view customization.

## Additional Information
- The fallback view ensures users can still interact with the availability matrix even without a team.
