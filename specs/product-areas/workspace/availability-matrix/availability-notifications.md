# Availability Notifications

## Story
- **IN ORDER TO** stay informed about team availability changes
- **AS** a user
- **I WANT TO** receive notifications for availability updates

## Acceptance Criteria
- Users can opt-in to receive notifications for availability changes.
- Notifications include the user, date, and status change.
- Notifications are sent via email or in-app.

## In-Scope
- Notification preferences in user settings.
- Backend logic for sending notifications.

## Out-of-Scope
- Push notifications (covered in a separate story).

## Additional Information
- Notifications should respect user preferences.
