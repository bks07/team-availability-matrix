# Availability Summary

## Story
- **IN ORDER TO** get an overview of team availability
- **AS** a user
- **I WANT TO** see a summary of availability statuses for the selected year

## Acceptance Criteria
- Users can view a summary of W/V/A statuses for the team.
- The summary is displayed as a table or chart.
- The data updates dynamically based on the selected year.

## In-Scope
- Summary table or chart in the matrix UI.
- Backend API for fetching summary data.

## Out-of-Scope
- Exporting summary data (covered in a separate story).

## Additional Information
- The summary should highlight weekends and public holidays.
