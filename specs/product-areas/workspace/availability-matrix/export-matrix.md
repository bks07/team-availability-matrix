# Export Availability Matrix

## Story
- **IN ORDER TO** analyze availability data offline
- **AS** a user
- **I WANT TO** export the availability matrix as a CSV file

## Acceptance Criteria
- Users can export the matrix for a selected year.
- The exported file includes all team members and their statuses.
- The file is formatted as a CSV.

## In-Scope
- Export button in the matrix UI.
- Backend endpoint for generating CSV files.

## Out-of-Scope
- Exporting data in other formats (e.g., Excel).

## Additional Information
- The CSV file should include headers for clarity.
