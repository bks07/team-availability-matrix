# Multi-Factor Authentication

## Story
- **IN ORDER TO** enhance account security
- **AS** a user
- **I WANT TO** enable multi-factor authentication for my account

## Acceptance Criteria
- Users can enable MFA via an authenticator app.
- The system requires a verification code during login if MFA is enabled.
- Users can disable MFA from their account settings.

## In-Scope
- MFA setup and verification process.
- Backend support for MFA tokens.

## Out-of-Scope
- SMS-based MFA (covered in a separate story).

## Additional Information
- MFA should be optional but recommended for all users.
