# Set Default Team

## Story
- **IN ORDER TO** streamline my workflow
- **AS** a user
- **I WANT TO** set a default team that is displayed immediately after login

## Acceptance Criteria
- Users can select a default team from the list of teams they are a member of.
- The selected default team is saved in the user's profile.
- The default team is displayed on login unless manually changed during the session.

## In-Scope
- Default team selection UI in user profile settings.
- Backend support for saving and retrieving the default team.

## Out-of-Scope
- Team creation or management.
- Notifications for team changes.

## Additional Information
- The default team setting should persist across sessions.
