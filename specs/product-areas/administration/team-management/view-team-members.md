# View Team Members

## Story
- **IN ORDER TO** see which employees belong to a team
- **AS** an administrator
- **I WANT TO** view a list of team members

## Acceptance Criteria
- Admins can select a team to view its members.
- The system displays a list of users with their names and roles.
- A message is displayed if the team has no members.

## In-Scope
- Team member listing interface.
- Backend API for fetching team members.

## Out-of-Scope
- Editing team details (covered in a separate story).

## Additional Information
- This feature is essential for managing team assignments.
