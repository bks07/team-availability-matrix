# Permission Usage Report

## Story
- **IN ORDER TO** analyze how permissions are being used within profiles
- **AS** an administrator
- **I WANT TO** generate a report of permission assignments and usage

## Acceptance Criteria
- Admins can generate a report of all permissions and their assignments within profiles.
- The report includes user names, profiles, and direct assignments.
- The report is downloadable as a CSV file.

## In-Scope
- Report generation interface.
- Backend logic for compiling report data.

## Out-of-Scope
- Real-time usage tracking (covered in a separate story).

## Additional Information
- The report should be filterable by permission and user.
