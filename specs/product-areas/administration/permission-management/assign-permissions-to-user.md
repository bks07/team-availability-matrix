# Assign Permissions to User

## Story
- **IN ORDER TO** control user access to system actions
- **AS** an administrator
- **I WANT TO** assign specific permissions to users

## Acceptance Criteria
- Admins can assign permissions to users via a dropdown or search interface.
- The system updates the user's permissions upon assignment.
- A success message is displayed upon successful assignment.

## In-Scope
- User permission assignment interface.
- Backend updates for user-permission relationships.

## Out-of-Scope
- Permission profiles (covered in separate stories).

## Additional Information
- Users can have multiple permissions assigned directly.
