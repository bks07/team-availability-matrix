# Assign Profile to User

## Story
- **IN ORDER TO** simplify user permission management
- **AS** an administrator
- **I WANT TO** assign permission profiles to users

## Acceptance Criteria
- Admins can assign profiles to users via a dropdown or search interface.
- The system updates the user's permissions based on the profile.
- A success message is displayed upon successful assignment.

## In-Scope
- User profile assignment interface.
- Backend updates for user-profile relationships.

## Out-of-Scope
- Direct permission assignments (covered in separate stories).

## Additional Information
- Users can have only one profile assigned at a time.
