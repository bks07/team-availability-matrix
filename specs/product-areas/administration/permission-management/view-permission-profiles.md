# View Permission Profiles

## Story
- **IN ORDER TO** understand the available profiles and their permissions
- **AS** an administrator
- **I WANT TO** view a list of all permission profiles in the system

## Acceptance Criteria
- Admins can view a list of profiles with their names and included permissions.
- The list is paginated for large numbers of profiles.
- A search bar allows filtering by name or included permissions.

## In-Scope
- Profile listing interface.
- Backend API for fetching profiles.

## Out-of-Scope
- Editing or deleting profiles (covered in separate stories).

## Additional Information
- This feature is essential for managing profiles effectively.
